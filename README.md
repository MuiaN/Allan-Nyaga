# Safari Adventures Kenya - Landing Page

This is a modern, responsive, and visually appealing landing page for a safari tour company, "Safari Adventures Kenya". The project is built with Next.js and styled with Tailwind CSS, showcasing various safari packages and services.

![Safari Adventures Kenya Screenshot](https://images.pexels.com/photos/16116697/pexels-photo-16116697.jpeg?auto=compress&cs=tinysrgb&w=1260&h=750&dpr=1)

- **Modern & Responsive Design**: A clean layout that looks great on all devices, from mobile phones to desktop screens.
- **Interactive UI Components**: Built with **shadcn/ui** for high-quality and accessible components.
- **Hero Section**: A stunning full-screen hero section with a compelling call-to-action.
- **Services Showcase**: A grid layout highlighting the company's key services with hover effects.
- **Featured Packages Carousel**: An interactive carousel to display featured safari packages, complete with custom navigation controls.
- **Contact & Booking Form**: A dedicated section for users to get in touch or request a custom safari quote.
- **Comprehensive Footer**: Includes navigation links, contact information, and social media icons.

## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Make sure you have Node.js and npm (or yarn/pnpm) installed on your machine.

- **Node.js** (v18.x or later recommended)
- **npm**
  ```sh
  npm install npm@latest -g
  ```

### Installation

1.  Clone the repository to your local machine:
    ```sh
    git clone https://github.com/your-username/safari-adventures-kenya.git
    ```
2.  Navigate into the project directory:
    ```sh
    cd safari-adventures-kenya
    ```
3.  Install the required NPM packages:
    ```sh
    npm install
    ```

### Running the Development Server

Once the dependencies are installed, you can run the development server:

```sh
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. The page will auto-update as you edit the files.
